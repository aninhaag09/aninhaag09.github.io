Apresentação

Este blog foi desenvolvido como um exercício prático de estrutura web. O objetivo é criar uma interface funcional utilizando apenas HTML e CSS, priorizando a legibilidade e a sobriedade visual.

A escolha cromática em preto e off-white reflete uma busca pela estética clássica dos antigos impressos, onde o conteúdo escrito é o protagonista. Aqui, pretendo registrar meus avanços acadêmicos e técnicos.

Interesses Técnicos

Atualmente, meu foco de estudo está concentrado em:

  • Desenvolvimento Front-end estrutural
  • Design de interfaces minimalistas
  • Lógicas de programação aplicadas

Créditos

Estrutura baseada no modelo de colunas proposto em sala de aula.
